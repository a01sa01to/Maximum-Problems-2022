mex (easy)

  • 実行時間制限: 1 sec
  • メモリ制限: 2 GB

問題文

数列 $A_1, A_2, ..., A_n$ が与えられます。この数列に含まれない、最小の非負整数を求めてください。

制約

  • $1 \le n \le 10^5$
  • $0 \le A_i \le 10^9$

入力

入力は以下の形式で標準入力から与えられる。

$n$
$A_1$ $A_2$ ... $A_n$

出力

数列に含まれない、最小の非負整数を求めてください。

入出力例

例1

入力

3
0 3 2

出力

1

非負整数は「0, 1, 2, 3, 4, ...」と続きますが、 0, 2, 3 は数列に含まれています。よって、 1 を出力します。

例2

入力

3
100 200 300

出力

0

非負整数なので、 0 を含むことに注意してください。

例3

入力

11
0 1 10 100 1000 10000 100000 1000000 10000000 100000000 1000000000

出力

2

End of Problem
